Text is the main way that teams visualize content and complete their work, so we’ve developed a scale that uses system fonts for Asana apps. This ensures that the UI is optimized to be highly legible and performs well on any device.

Font stacks

Use sans-serif fonts for most of our type, the exception being when you want to display code then you should defer to monospace fonts.

Usage

Keeping consist and sticking to logical hierarchies ensures that elements in the UI are clear and easily recognizable when scanning the page. Text sizes, styles and layouts were selected to balance content and UI and to foster familiarity. Type sizes are specified with sp (scaleable pixels) to enable large type modes for accessibility.
